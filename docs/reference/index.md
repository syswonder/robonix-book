# 参考

本章是 Robonix 的机读参考，分两部分：

- [能力约定参考](contracts.md) 与 [ROS IDL 参考](idl.md)——由 `rbnx docs` 从 `capabilities/` **自动生成**。这两页是能力约定与数据结构的权威来源；[接口目录](../interface-catalog/index.md) 解释用途、调用方式和组合关系，原始结构以这里为准。本网站使用的 Robonix 源码修订号记录在 Book 仓库根目录的 `ROBONIX_SOURCE_REVISION`。
- [代码 API](api.md)——各 Rust crate / Python 包的 rustdoc / Sphinx 入口。

> 自动生成的两页（[能力约定参考](contracts.md) / [ROS IDL 参考](idl.md)）请勿手改——它们由 `rbnx docs` 生成。
